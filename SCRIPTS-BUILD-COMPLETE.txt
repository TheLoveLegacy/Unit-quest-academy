HOW TO USE
================================================================================

FOR ADAM (Manager):
1. Open Dashboard → Scripts
2. Browse or search for scripts
3. Click script to view details
4. See Mom's feedback below content
5. Update script in Supabase if needed
6. Resolve feedback with notes
7. Update script status when ready

FOR MOM (Reviewer):
1. Open Dashboard → Scripts
2. Find script to review
3. Click "+ Add Feedback"
4. Select feedback type (comment, suggestion, approval, revision)
5. Write feedback
6. Submit - Adam sees it immediately
7. Check back for Adam's response

FOR UQA PIPELINE USERS:
1. Open UQA Pipeline view
2. See script status next to each unit
3. Click "View" to jump to script
4. Review content and feedback
5. Return to pipeline to continue
